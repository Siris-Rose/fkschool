# ﻿学校体温自动上报

---

最近武汉肺炎形势严重，但各方都还在搞形式主义。。比如学校这个天天上报体温，无语。

闲来无事拿着练习python的借口写了个自动化，自动填写体温等信息并提交。

使用方法
---

需要python3, selenium以及chromedriver。

我都Push上来了。。除了↑

运行之前自己修改头部的初始信息，将××××××替换成自己的信息即可。默认当天填写情况和昨天一致，~~心理状态为正常。~~
