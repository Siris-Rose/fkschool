# ﻿学校体温自动上报

---

最近武汉肺炎形势严重，但各方都还在搞形式主义。。比如学校这个天天上报体温，无语。

闲来无事拿着练习python的借口写了个自动化，自动填写体温等信息并提交。

## 免责声明

此脚本只做方便同学打卡之用

会默认与昨日信息相同，且自动生成一正常体温填入。

如有体温不正常或所在地发生变化请立即停用此脚本并如实上报。
